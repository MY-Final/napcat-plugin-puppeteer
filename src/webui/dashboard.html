核心功能接口

POST /puppeteer/screenshot

通用截图接口，支持 URL、本地文件、HTML 字符串

{
  "file": "https://example.com",        // 必填: URL / HTML字符串 / 文件路径
  "file_type": "auto",                  // 可选: auto | htmlString | url | file
  "data": { "name": "World" },          // 可选: 模板数据 (Handlebars)
  "selector": "body",                   // 可选: 截图元素选择器
  "type": "png",                        // 可选: png | jpeg | webp
  "quality": 80,                        // 可选: 图片质量 (1-100, 仅 jpeg/webp)
  "encoding": "base64",                 // 可选: base64 | binary (默认 base64)
  "fullPage": false,                    // 可选: 是否截取全屏
  "omitBackground": false,              // 可选: 是否隐藏默认背景(透明背景)
  "setViewport": {                      // 可选: 视口设置
    "width": 1280,
    "height": 800,
    "deviceScaleFactor": 1
  },
  "waitForSelector": ".content",        // 可选: 等待元素出现
  "waitForTimeout": 1000                // 可选: 等待毫秒数
}
POST /puppeteer/render

HTML 模板渲染专用接口 (是 /screenshot 的别名，更语义化)

{
  "html": "<h1>Hello {{name}}</h1>",  // 必填: HTML 模板字符串
  "data": { "name": "World" },          // 可选: 模板数据
  "selector": "body",                   // ...其他参数同 /screenshot
  "encoding": "base64"
}
GET /puppeteer/screenshot

快速 URL 截图 (Query 参数)

url: https://example.com (必填)
selector: body
fullPage: true | false
type: png | jpeg | webp
raw: true (直接返回图片流)
